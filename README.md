# CC Power Toolkit

**Stop burning context mid-session and blowing your monthly Claude Code budget.**

A Claude Code plugin that adds real-time context monitoring, API spend tracking,
and one-command session summarization to your workflow.

---

## The problems this solves

- You're deep in a coding session and Claude suddenly loses context — all your decisions, history, and progress gone
- You hit your monthly API limit (or company-allotted balance) two weeks into the month
- You have no idea how much of your context window is left until it's too late

## What's included

**Context Sentinel** — Monitors your context window usage and warns you at 70% and 85% with specific action recommendations. Never get blindsided mid-session again.

**Budget Guard** — Tracks cumulative API spend against a daily or monthly limit you set. Warns you before you hit it, not after.

**Smart Summarizer** — One command compresses your entire session into a structured handoff note. Paste it into a fresh session and continue exactly where you left off.

